# Lucero TRP

Backend API para la aplicación de transporte Lucero TRP.

## Stack
- Node.js + Express
- MongoDB + Mongoose
- TypeScript
- Socket.IO (WebSocket)

## Deploy
- Backend: Render
- Database: MongoDB Atlas

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm start
```

## Seed (Crear usuario admin)

```bash
npm run seed
```
